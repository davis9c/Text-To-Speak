# TTS Documentation

## Overview

This repository contains a production-ready Announcement Server roadmap.

Recommended documentation structure:

-   README.md
-   docs/ARCHITECTURE.md
-   docs/API.md
-   docs/TTS.md
-   docs/WORKER.md
-   docs/ZONES.md
-   docs/ROADMAP.md
-   docs/DECISIONS.md

## Purpose

The project is designed as a modular, production-ready Public Address
(PA) and Text-to-Speech server for Windows using Python, FastAPI,
offline TTS (Piper), queue management, multi-zone playback, scheduling,
monitoring, and Windows Service deployment.

## Documentation Philosophy

-   README.md: User guide
-   ARCHITECTURE.md: System architecture
-   API.md: REST API reference
-   TTS.md: TTS engine design
-   WORKER.md: Queue and worker design
-   ZONES.md: Multi-zone playback
-   ROADMAP.md: Development phases
-   DECISIONS.md: Architecture Decision Records (ADR)
